# Build

## Prerequisites
- Rust
- Node.js
- pnpm

## How to build
* Clone this repository
```sh
git clone https://github.com/farhannz/kabua.git && \
cd kabua
```
* Install node dependencies
```sh
pnpm install
```
* Start developing
```sh
pnpm tauri dev
```
* or to build the app
```sh
pnpm tauri build
```

## Known limitations
- No OTP Support
- No Steam Support
- No Captcha Solver
